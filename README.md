# 06HeroesApp

Paquete usado para backend (local): https://www.npmjs.com/package/json-server

### Dev

1. Ejecurat `npm install`
2. Levantar backend `npm run backend` (definido en package.json)
3. Ejecurat la app `npm start` o bien ``ng serve -o`
